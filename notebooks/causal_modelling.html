Intro: Causal models as Bayesian Networks¶

  • A causal model has the same form as a probabilistic Bayesian network. It consists of a directed acyclic graph over the random variables in the domain.
  • The model asserts that each variable X is governed by a causal mechanism that (stochastically) determines its value based on the values of its parents. That is, the value of $X$ is a (stochastic) function of the values of its parents.
  • For a node $X$ and its parents $\textbf{U}$, the causal model has a stochastic function from the values of $\textbf{U}$ to the values of $X$. In other words, for each value u of $\textbf{U}$, it specifies a distribution over the values of $X$.
  • We assume that causality flows in the direction of the edges, so that $X$s value is actually determined via the stochastic function implied by $X$s CPD.
  • The assumption that CPDs correspond to causal mechanisms forms the basis for the treatment of intervention queries. When we intervene at a variable $X$, setting its value to $x$, we replace its original causal mechanism with one that dictates that it take the value $x$. This manipulation corresponds to replacing $X$s CPD with a different one, where $X = x$ with probability 1, regardless of anything else.

Multilated networks¶

  • With the explanation of the mechanisms describing how a causal network works we can build a first working definition of a causal model. Which takes into accound a multilated network.

  • $\textbf{Def}$: Given a bayesian network $\mathcal{B}$ let the mutilated network $\mathcal{B}_{Z=z}$ be obtained by removing all incoming edges into $Z$.

  • Thus given a causal model $P_{Cs}$ the solution to the causal query of the form $P_{Cs}(Y|\mathrm{do}(z),x) = P_{Cs_{Z=z}}(Y|x)$ which suggests we can find the solution from the corresponding mutilated network.

Augmented networks¶

  • WHEN WE INTERVENE WE HAVE A NEW DISTRIBUTION!!!! LOOKING FOR A TRANSFORMATION BETWEEN $P$ and new $P'$. WE WANT TO ANSWER KNEW QUERIES.

  • Simplest example, suppose some variable $X_i$ is forced to take on a value say $X_i = x_i'$. In some sense there is an external force $F_i$ which sets the variable $X_i = x_i'$.

  • The effect of an atomic intervention $\mathrm{do}(X_i = x_i')$ is encoded by adding a link $F_i \rightarrow X_i$ where $F_i$ is a new variable taking on values $\{\mathrm{idle}, \mathrm{do}(x_i')\}$ where $x_i'$ ranges over the domain of $X_i$.

  • The new parent of $X_i$ in the augmented graph $G'$ is represented by $Pa(X_i)' = Pa(X_i) \cup \{F_i\}$. We can place an arbitrary prior over $F_i$ and we will see this is practice is irrelevant.

Truncated Factorisation: What are we interested in?¶

  • What we are now really interested in is the conditional probability distribution given an intervetion $F_i = do(x_i)$.

  • That is we are interested in the following $$P_{x_i'}(x_1,\ldots,x_n) = P'(x_1,\ldots,x_n|F_i = do(x_i')).$$

  • Lets have a quick summary of what we have so far. Firstly we have a causal model $\langle G,P \rangle$ and an augmented causal model $\langle G',P' \rangle$. We are interested in the conditional distribution $P_{x_i'}'$ which represents the effect of the intervention $do(x_i')$.

  • What we want to do is get the conditional distribution representing the effect of the intervention in terms of the original.

  • We can think of intervening at $X_i$ as removing the dependance on the parents of $X_i$ in $G$. Therefore all we have to do is remove the factor $P(x_i|pa_i)$ from the distribution of $P$.

  • This gives us the following form for $P_{x_i'}$ interms of the original distribution $P$ namely
$$\begin{cases} P'_{x_i'}(x_1,\ldots,x_n) = \frac{P(x_1,\ldots,x_n)}{P(x_i|pa_i)}\mathrm{\;if\;}x_i=x_i'\\ 0\mathrm{\;if\;} x_i \neq x_i'\\ \end{cases} $$

A calculus of actions¶

  • Let $X,Y,Z$ and $W$ be four arbitrary disjoint sets of nodes in the DAG $G$. We say that $X$ and $Y$ are independent given $Z$ in $G$, denoted $(X \perp Y|Z)_{G}$, if the set $Z$ d-seperates $X$ from $Y$ in $G$.

  • We now define the notion of a mutilated graph where we delete some edges. Let $G_{\overline{X}}, G_{\underline{X}}$ obtained by deleting the edges pointing to resp pointing from $X$.

Rule 1¶

(Insertion/Deletion of observations) Consider the augmented graph $G'$ by adding the external variable $F_{X}$ corresponding to the variable $do(x)$ then $P'(y|do(x),z,w) = P'(y|do(x),w)$ if $(Y \perp Z|X,W)_{G_{\overline{X}}}$.

Justification: Recall in our augmented graph if we are given $do(x)$ then we have effectively broken the link between $X$ and its parents in G' and so testing whether $(Y \perp Z|do(x),W)$ in $G'$ is equivalent to testing whether $(Y \perp Z|X,W)$ in $G_{\overline{X}}$. Essentially since we have a deterministic cpd on $X$ knowing $do(x)$ is the same as knowing $X=x$.

What this says is that if by removing the parents of $X$ there is no path from $Z$ to $Y$ then we know $Z$ cannot influence $Y$ by d-seperation given $X$ (in this case obtained from the do(x) operator) and $W$.

Rule 2¶

Action/observation exchange $P(y|do(x),do(z),w) = P(y|do(x),z,w)$ if $(Y \perp Z|X,W)_{G_{\overline{X},\underline{Z}}}$
